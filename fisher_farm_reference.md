## 1. `farm` 変数（農場全体）
`farm` は、農場の全 64 区画（プロット）を管理する **2次元リスト（行列）** です。

* **構造**: 8行 × 8列 の 2次元配列。
* **指定方法**: `farm[行番号][列番号]` 
    * 行番号（y）：0 ～ 7
    * 列番号（x）：0 ～ 7
    * 例: `farm[0][0]` は一番左上、`farm[7][7]` は一番右下の区画を指します。

---

## 2. `Plot` オブジェクト（区画のデータ）
`farm` の各要素（例: `farm[0][0]`）は、`Plot` という型のオブジェクトになっています。
それぞれのオブジェクトは、その場所の「環境」や「栽培状況」のデータを持っています。

### 属性（プロパティ）一覧
Python コード内で `オブジェクト名.属性名` と記述することで、データの読み書きができます。

| 属性名 | データ型 | 説明 | 書き換え |
| :--- | :--- | :--- | :--- |
| `.variety` | `str` | **品種名**。'A', 'B', 'C' のいずれかを代入します。 | **可能** |
| `.is_sunny` | `bool` | **日当たり条件**。`True`（日向）または `False`（日陰）。 | 不可 |
| `.is_water` | `bool` | **地形条件**。`True`（水辺）または `False`（陸地）。 | 不可 |
| `.yield_val` | `float` | **収穫量**。収穫実行後、数値（0.0〜）が格納されます。 | 不可 |

---

## 3. 農場の環境分布
農場（8×8マス）の環境条件は、あらかじめ以下のように決まっています。

* **日当たり (`is_sunny`)**
    * 0 ～ 3 行目（上半分）：`True` (☀️ 日向)
    * 4 ～ 7 行目（下半分）：`False` (☁️ 日陰)
* **水辺 (`is_water`)**
    * 0 ～ 1 列目（左端）：`True` (💧 水辺)
    * 2 ～ 7 列目（右側）：`False` (🏞️ 陸地)

---

## 4. プログラミング例

### 品種を植える（実験の計画）
「計画」タブのエディタに記述します。

```python
# 左上の区画（0行0列）に品種'A'を植える
farm[0][0].variety = 'A'

# 全ての「日向」の区画に品種'B'を植える
for y in range(8):
    for x in range(8):
        if farm[y][x].is_sunny == True:
            farm[y][x].variety = 'B'
```
